# Del grosso Adam - Portfolio

## À propos de ce Portfolio

Ce portfolio a été conçu pour présenter mes compétences, projets et expériences professionnelles en tant que développeur. Ce n'est qu'une version temporaire qui a pour but d'évoluer dans le temps au fil de mes expériences et progression de mes compétences.

## Pourquoi un Portfolio ?

- Une manière professionnelle de présenter mon travail
- Augmente ma visibilité et présence en ligne
- Meilleures chances d'opportunités professionnelles

---

## Démarrage Rapide

Pour exécuter ce projet localement pour le développement et les tests :

### Prérequis

Vous aurez besoin de Git, Node.js et NPM installés sur votre machine.

### Installation

```bash
# Cloner le dépôt
$ git clone https://github.com/LeFloppa/Portfolio.git

# Se déplacer dans le dossier du projet
$ cd Portfolio

# Installer les dépendances
$ npm install

# Lancez le projet localement
$ npm start
```
